Display and Position

Property Description Values
z-index Sets the z-order of a positioned element and its descendants auto or an integer
display Specifies how an element is shown on a web page block, inline, flex, etc.
position Specifies the type of positioning method used for an element static, relative, fixed, absolute, sticky
float Specifies how an element should float. left, right, none, inherit
clear Specifies what should happen with the element that is next to a floating element none, left, right, both, inherit
